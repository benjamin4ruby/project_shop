h1. Project Shop

This will be our example ruby-on-rails application to get started becoming a rubyist.
Feedback about coding-style, security etc. is always welcome.

h2. Features planned:
* A product can have many categories, a photo, a price, a title, and a non-specified set of other properties
* Categories can have 1 (several?) parent category
* Search (most relevant products first)
* Shopping Cart
* Checkout
* Register of Users
* Admin users can:
** Import/Export of XML
** Edit Product
** View Orders
** Manage users
